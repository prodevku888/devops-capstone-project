# DevOps Capstone Project

## Judul Proyek
Build an Automated CD DevOps Pipeline Using Tekton and OpenShift

## Deskripsi
Proyek ini bertujuan untuk membangun pipeline Continuous Delivery (CD) otomatis menggunakan Tekton di lingkungan OpenShift. Pipeline ini akan mengotomatisasi proses build, test, dan deployment aplikasi secara end-to-end sesuai dengan praktik DevOps modern.

## Fitur Utama
- Otomatisasi build dan deployment aplikasi
- Integrasi dengan OpenShift
- Pipeline CI/CD menggunakan Tekton
- Pengujian otomatis
- Monitoring dan notifikasi

## Struktur Direktori
- `devsecinfra/` : Infrastruktur dan keamanan
- `github/`      : Konfigurasi GitHub
- `deploy/`      : Manifests deployment
- `tekton/`      : Pipeline Tekton
- `tests/`       : Pengujian aplikasi
- `bin/`         : Skrip setup dan utilitas

## Cara Menjalankan
1. Clone repository ini
2. Jalankan skrip setup di `bin/setup.sh`
3. Ikuti instruksi pada pipeline Tekton di folder `tekton/`
